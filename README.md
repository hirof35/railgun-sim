# レールガン物理シミュレーター (Node.js GUI)

Node.jsとHTML5 Canvasを使用した、電磁気加速砲（レールガン）の挙動を視覚的に理解するための簡易物理シミュレーターです。
スライダーを使って電流や弾体の質量をリアルタイムに調整し、ローレンツ力による加速から自由飛行に移る一連のプロセスをアニメーションで確認できます。

## 💡 特徴

- **物理数式に基づいた演算**: インダクタンス勾配、ローレンツ力、加速度の関係性をシミュレート。
- **直感的なGUI**: 印加電流（A）や弾体質量（g）をスライダーで自由に変更可能。
- **軽量動作**: 外部依存ライブラリ（npmパッケージ等）は一切不要。Node.jsの標準機能のみで動作します。

## 🛠️ 動作環境

- Node.js (v14以上推奨)
- モダンなWebブラウザ（Google Chrome, Microsoft Edge, Safari, Firefoxなど）

## 📂 ディレクトリ構成

```text

🚀 使い方（クイックスタート）ファイルの準備任意のフォルダを作成し、本リポジトリの server.js と index.html を配置します。サーバーの起動ターミナル（またはコマンドプロンプト）を開き、ファイルがあるディレクトリに移動して以下のコマンドを実行します。Bashnode server.js
ブラウザでアクセスサーバーが起動するとターミナルにURLが表示されます。ブラウザを開き、以下にアクセスしてください。http://localhost:3000シミュレーションの実行画面左側のスライダーで「印加電流」や「弾体の質量」を調整します。[砲撃開始 (FIRE)] ボタンを押すと、弾体（球）が加速して発射されます。🧮 物理モデルについて本シミュレーターは、以下の基本的な電磁気学および力学の数式をベースに計算を行っています。磁気インダクタンス勾配 ($L'$)$$L' = \frac{\mu_0}{\pi} \ln\left(\frac{d}{r}\right)$$( $\mu_0$: 真空の透磁率, $d$: レール間距離, $r$: レール半径 )ローレンツ力 ($F$)$$F = \frac{1}{2} L' I^2$$( $I$: 印加電流 )運動方程式 ($a$)$$a = \frac{F}{m}$$( $m$: 弾体質量 )注意: 本シミュレーターは、物理現象の基本的な概念理解を目的とした「理想状態」のモデルです。実際の環境で発生する摩擦抵抗、空気抵抗、ジュール熱によるレールの電気抵抗の変化、プラズマの発生などの損失要素は簡略化のため省略されています。
